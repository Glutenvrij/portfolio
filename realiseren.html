Leerdoel 7

Tijdens mijn stage wil ik code hebben geschreven en opgeleverd van een lopende taak uit de backlog.

Gedurende mijn stage wil ik ervaring op doen van de gebruikte code om hiervoor meer werkzaamheden op te kunnen pakken en af te kunnen ronden. Zowel als om meer codeer ervaring op te doen heb ik dit leerdoel opgezet.


Inleiding

Voor een demo zal een bestaand scherm aangepast worden zodat de klant een voorbeeld van het eindresultaat heeft, dus hoe een scherm eruit komt te zien wanneer het product opgeleverd is.

De vraag is om teksten op een scherm aan te passen en na de demo weer terug te zetten. Dit is natuurlijk geen goede oplossing voor elke nieuwe demo. Hierom zal deze tekst in dit scherm dynamisch gemaakt worden zodat de bedrijfsnaam op dit scherm in de toekomst makkelijk aangepast kan worden voor andere klanten.

Reflectie

In de eerste maand van mijn stageperiode heb ik deze taak op mogen pakken. Door in de code te duiken heb ik een goed idee gekregen van de technieken en structuur waardoor ik andere werkzaamheden makkelijker kon oppakken. Dit heeft ook geholpen met research naar mijn stageopdracht.

Het proces van werken is gegaan zoals ik het gepland had en zou ik niet anders hebben gedaan. Wel had ik beter naar de vertalingen kunnen kijken zodat er geen extra commits gemaakt hoefde te worden door kleine foutjes. Mijn stagebegeleider zette de nadruk op handmatig committen van mijn werk via de console om dit te leren, waardoor ik minder overzicht had op de aanpassingen die ik had gemaakt, vergeleken met het inzicht van een git client dat ik gewend was. Hierdoor heb ik wel een beter idee gekregen van de werking van commits.

Door het gehele proces heen kon ik makkelijk vragen stellen aan collega's als ik dit had over bijvoorbeeld code, of waar ik iets in de bestanden kon vinden. Als feedback kreeg ik om meer gedetailleerde testomschrijvingen te schrijven om het werk makkelijker te maken voor de testers.

Bij het bespreken van de taak en vragen stellen tussendoor ben ik bezig geweest met zowel Front-end, Backend, de Database en heb ik kennis opgedaan van vertalingtools en het softwareontwikkelproces van mijn stagebedrijf. Door het proces van implementeren, testen, implementeren van feedback en oplevering vind ik met deze taak dit leerdoel afgerond te hebben.
